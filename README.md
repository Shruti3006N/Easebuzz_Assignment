# QA Engineer Assignment — Complete Solution

A complete, well-structured QA automation assignment covering API testing,
Pytest, Postman, Playwright, and QA leadership strategy.


## ⚙️ Setup Instructions

### 1. Clone the repository
```bash
git clone https://github.com/<your-username>/qa_assignment.git
cd qa_assignment
```

### 2. Create and activate a virtual environment
```bash
# macOS / Linux
python3 -m venv venv
source venv/bin/activate

# Windows
python -m venv venv
venv\Scripts\activate
```

### 3. Install dependencies
```bash
pip install -r requirements.txt
```

### 4. Install Playwright browsers (for Task 4 only)
```bash
playwright install chromium
```
